Ocedata is a package for the R statistical language that provides oceanographic datasets to support the Oce package.

The provided datasets are as listed below. To learn more about a given one, type e.g. help("airHalifax", "ocedata") in an R console. The manual is available within R and online.

  • beaufort.rda
