Ocedata is a package for the R statistical language that provides oceanographic datasets to support the Oce package.

The provided datasets are as listed below.

* RRprofile * airHalifax * beaufort * buoy * coastlineWorldFine * coastlineWorldMedium * drag * endeavour * geosecs235 * giss * landsat * levitus * munk * oceans * papa * redfieldNC * redfieldNP * redfieldPlankton * riley * schmitt * secchi * topo2 * turbulence * wilson
